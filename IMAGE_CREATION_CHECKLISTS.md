## DESIGN STYLE GUIDE

### Consistent Visual Language

All images should follow these principles:

**Color Usage:**
- [ ] Primary: #C856D7 (Magenta)
- [ ] Dark: #1F0F3D (Navy Purple)
- [ ] Accent: #D4A574 (Gold)
- [ ] Light: #8B70B0 (Light Purple)
- [ ] Text: #7A6099 (Muted Purple)
- [ ] White: #FFFFFF (Clean)

**Style Consistency:**
- [ ] Modern, minimal aesthetic
- [ ] Flat design or subtle 3D
- [ ] Generous whitespace
- [ ] Clear typography hierarchy
- [ ] Smooth curves and rounded corners
- [ ] Gradient backgrounds (optional)
- [ ] No dropped shadows unless intentional
- [ ] Professional polish

**Design Trends to Incorporate:**
- [ ] Glassmorphism elements (in applicable designs)
- [ ] Geometric shapes
- [ ] Abstract representations
- [ ] Handmade or organic feel
- [ ] Motion-ready (simple, clean lines)
- [ ] Accessible colors (good contrast)

**Design Trends to Avoid:**
- [ ] Overly complex details
- [ ] Trendy but dated styles
- [ ] Inconsistent line weights
- [ ] Clashing colors
- [ ] Low-quality or pixelated assets
- [ ] Unnecessary 3D effects
- [ ] Busy patterns
- [ ] Inconsistent branding

## TOOLS & RESOURCES FOR CREATION

### Design Tools
- **Figma** (Free): Best for web design
  - Built-in responsive design features
  - Easy to export to multiple formats
  - Free tier sufficient for this project

- **Adobe Creative Suite** (Paid): Professional
  - Photoshop (raster images)
  - Illustrator (vector graphics)
  - XD (UI/UX design)

- **Canva Pro** ($120/year): Templates
  - 500px+ templates
  - Easy customization
  - Built-in optimization

- **Adobe Express** (Free): Quick graphics
  - Templates for social/web
  - Generative fill (AI)
  - Easy export

### AI Image Generation (for inspiration or base)
- **Midjourney**: High-quality, requires payment
- **DALL-E 3**: Good quality, free credits
- **Stable Diffusion**: Open-source, free
- **Adobe Firefly**: Integrated with Creative Suite

### Icon Libraries
- **Feather Icons** (Free, SVG)
- **Heroicons** (Free, SVG)
- **IcoMoon** (Free, customizable)
- **Font Awesome** (Free tier)

### Stock Resources (Paid - for reference)
- **Unsplash** (Free): High-quality photos
- **Pexels** (Free): Free stock images
- **Pixabay** (Free): Free images/videos
- **Dribbble** (Free): Design inspiration
- **Behance** (Free): Professional showcases

### Optimization Tools
- **TinyPNG** (tinypng.com): Free compression
- **Squoosh** (squoosh.app): Google's optimizer
- **ImageMagick**: Command-line batch processing
- **Sharp**: Node.js library for processing
- **SVGO**: SVG optimizer (command-line)

## Quality Assurance Checklist

Before finalizing each image:

- [ ] Dimensions are exact (no scaling)
- [ ] File size is within limits
- [ ] Format is correct (WebP primary, JPG fallback)
- [ ] Colors match brand palette
- [ ] No compression artifacts
- [ ] Looks good on all screen sizes
- [ ] Alt text is descriptive
- [ ] File named correctly and in right folder
- [ ] Image is professional quality
- [ ] Consistent with other images

---

## Final Notes

- **Consistency is key**: All images should feel like they belong together
- **Quality over quantity**: One great image beats 3 mediocre ones
- **Optimization matters**: File sizes impact page speed
- **Test thoroughly**: Verify on mobile, tablet, and desktop
- **Get feedback**: Have someone review before finalizing
- **Version control**: Keep original PSD/Figma files for future edits
