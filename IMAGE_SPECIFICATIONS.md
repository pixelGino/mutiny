## 6. WEB OPTIMIZATION BEST PRACTICES

### Image Format Selection
```
- Photos/Complex: WebP (primary), JPG (fallback)
- Icons/Simple graphics: SVG or PNG with transparency
- Gradients/Illustrations: WebP or PNG
- Client logos: SVG or PNG (high quality)
```

### File Size Guidelines
```
- Hero images: 150KB max
- Card images: 80KB max
- Logos: 50KB max
- Icons: 30KB max
```

### Responsive Image Implementation
Include different sizes for different devices:
```html
<picture>
  <source srcset="image-1200.webp" media="(min-width: 1024px)" type="image/webp">
  <source srcset="image-800.webp" media="(min-width: 768px)" type="image/webp">
  <source srcset="image-400.webp" media="(max-width: 767px)" type="image/webp">
  <img src="image-fallback.jpg" alt="Description">
</picture>
```

### File Naming Convention
```
- service-digital-strategy.webp
- about-values-innovation.png
- case-study-mozilla.png
- client-logo-microsoft.png
- hero-ecosystem-1200.webp
```

```html

  <source srcset="images/hero-ecosystem.webp" type="image/webp">

<!-- Service Card Image -->
```
